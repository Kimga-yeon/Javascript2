자바스크립트 개요

스크립트 언어란?

기본 프로그램의 동작을 사용자 요구에 맞게 수행되도록 해주는 용도의 언어.
매우 빠르고, 배우기 쉽고, 소스코드 짧으며 파일을 이용한 상호작용이 가능하도록 고안되었음.

자바 스크립트란?

웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어.
ECMA 스크립트 표준을 따르는 대표적인 웹 기술이다.

자바스크립트 작성 방법

            소스코드는 함수 단위로 작성하고, 
            이벤트 속성을 사용하여 함수를 구동되게 함.

            이벤트 속성은 태그마다 차이가 있음.

            on이벤트명 = "실행할 함수명([전달값(매개변수)]);"

inline 방식

태그에 직접 간단한 소스코드를 작성해서 실행되게 하는 방법

internal 방식

소스코드를 script 태그 영역에 작성하고 작성된 함수를 태그에서 호출하는 방법.
script 태그는 head, body 태그에 작성 가능함.

external 방식

별도의 js파일을 이용하여 script내용을 따로 작성하고 가져다 사용하는 방법.
