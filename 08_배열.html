배열

자바스크립트의 배열이란?

자바스크립트에서는 별도의 자료형 지정이 없기 때문에
변수가 나열되어있는 배열도 자료형 관계없이 모든 값을 담을 수 있음.
--> 자바의 컬렉션과 비슷
+ 자바스크립트 배열은 별도의 크기 지정이 없고, 크기가 알아서 증가하는 특징이 있음.

toString(), join() : 배열을 결합하여 문자열로 반환

toString()은 배열을 문자열로 합칠 때 요소 사이에 ,(콤마)를 추가하여 합침
join()도 기본적으로 toString()과 같지만 매개변수로 구분자를 지정할 수 있음.

sort() : 배열을 오름차순으로 정렬
reverse() : 배열의 순서를 뒤집음

push(): 배열의 제일 뒤에 요소 추가
pop() : 배열의 제일 뒤에 요소를 꺼내옴.(배열 자체에서 요소가 삭제됨.)

shift(): 배열의 제일 앞에 요소 제거 ( pop()의 반대 동작 )
unshift : 배열의 제일 앞에 요소 추가 (push()의 반대 동작)

slice() : 배열의 특정 위치를 선택해서 잘라냄.
splice() : 배열의 특정 위치의 요소를 잘라내고 새로운 요소를 추가
