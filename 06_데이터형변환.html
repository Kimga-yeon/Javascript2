문자열과 숫자의 '+'연산

숫자 + 숫자 = 숫자
숫자 + 문자열 = 문자열
숫자 + 숫자 + 문자열 = 숫자가 먼저 계산되고, 뒤에 문자열과 합쳐짐
문자열 + 숫자 + 숫자 + 숫자 = 다 이어진 문자열 (Java와 똑같음)

강제 형변환

자바스크립트에서 문자열 -> 숫자로 변환하는 방법
1) parseInt() : 숫자로 이루어진 문자열 값을 정수로 변환
2) parseFloat() : 숫자로 이루어진 문자열 값을 실수로 변환
3) Number() 내장 객체 : 매개 변수로 숫자형태 문자열을 전달하면 해당 숫자가 반환됨.
